# REQUIREMENTS.md - 要件定義

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| サイト名 | HYデザイン |
| 種別 | ポートフォリオ / コーポレートサイト |
| 事業内容 | ホームページ制作、Web制作、パソコンサポート |
| 活動拠点 | 神奈川県 湘南・茅ヶ崎エリア |
| 対応エリア | 湘南・茅ヶ崎を中心に都内・地方も対応 |
| 事業形態 | フリーランス（個人事業主） |

## 2. ターゲットユーザー

### メインターゲット
- 湘南・茅ヶ崎エリアの **中小企業**
- **個人事業主**（飲食店、美容室、整体院、教室 etc.）
- 初めてホームページを作りたい人

### ユーザーの課題
- ホームページが欲しいが何から始めればいいかわからない
- 制作会社は料金が高く、敷居が高い
- 地元で気軽に相談できる人が欲しい
- 制作後の更新・修正も頼みたい

## 3. サイトの目的

1. **信頼獲得** - フリーランスとしてのスキルと実績を見せる
2. **集客** - SEO・口コミ経由でお問い合わせを獲得
3. **差別化** - 個人ならではの柔軟さ・低コストをアピール
4. **お問い合わせ誘導** - 全ページからContactへスムーズに導線

## 4. ページ構成

### 4.1 トップページ (`/`)
- ヒーローセクション（キャッチコピー + CTA）
- 5つの強み（HYデザインが選ばれる理由）
- サービス概要
- 制作実績プレビュー（3件程度）
- 料金表
- お問い合わせCTA

### 4.2 About (`/about`)
- 自己紹介・プロフィール
- 経歴・スキル
- 想い・理念（なぜこの仕事をしているか）
- 活動拠点の紹介

### 4.3 Service (`/service`)
- サービス一覧と詳細説明
  - ホームページ制作
  - LP（ランディングページ）制作
  - WordPress構築
  - サイト更新・保守
  - パソコンサポート
- 制作の流れ（ヒアリング → デザイン → 開発 → 公開）
- 料金プラン詳細

### 4.4 Works (`/works`)
- 制作実績一覧（カード形式）
- カテゴリフィルター（業種別）
- 各実績の詳細（スクリーンショット、概要、使用技術）

### 4.5 Contact (`/contact`)
- お問い合わせフォーム
  - お名前（必須）
  - メールアドレス（必須）
  - 電話番号（任意）
  - お問い合わせ種別（セレクト）
  - ご相談内容（テキストエリア）
- 電話番号・メールアドレス表示
- 対応エリアの説明

## 6. 非機能要件

| 項目 | 要件 |
|------|------|
| パフォーマンス | Lighthouse スコア 90+ |
| ブラウザ対応 | Chrome, Safari, Edge, Firefox（最新2バージョン） |
| アクセシビリティ | WCAG 2.1 AA 準拠を目指す |
| SEO | 構造化データ、OGP設定 |
| セキュリティ | フォームのバリデーション、XSS対策 |

## 7. HYデザインの強み（コンテンツ方針）

サイト全体で以下のメッセージを伝える：

1. **成果起点のWeb設計** - 見た目だけでなく、集客・売上につながるサイト設計
2. **中小企業・個人事業主特化** - 大手にはない、小規模ビジネスに寄り添った提案
3. **地域密着・柔軟対応** - 湘南・茅ヶ崎エリアで直接会って打ち合わせ可能、細かい要望にも対応
4. **伴走型サポート** - 制作して終わりではなく、公開後も一緒に育てていくパートナー
5. **ワンストップ対応** - 企画・デザイン・制作・運用・保守まで一括でお任せ
